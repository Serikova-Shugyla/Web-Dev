# Online Store - Angular

This project was created as part of Web Development Lab 4.

## Project Description

This Angular application displays a catalog of smartphones from kaspi.kz.

Features:
- 10 real smartphones
- Product image gallery (3 images per product)
- WhatsApp & Telegram share functionality
- Responsive grid layout
- TypeScript interface

---

## How to Run

1. Install dependencies:

npm install

2. Run development server:

ng serve

3. Open in browser:

http://localhost:4200
